JSON API

Name Description
basic.json basic info.
datasource.json DatsSourceStat List
datasource-{id}.json DatasourceStat for id={id},id is DatsSourceStat.ID
Example: druid/datasource-1.json
sql.json

SqlStat info List.

orderBy=[ID|ExecuteCount|...|](every property without histogram in sqlstat object)

orderType=[asc|desc] default is desc

page=page count start with 1

perPageCount=item count in one page,default is IN
sql-{id}.json SqlStat for id={id} ,id is SqlStat.ID
Example: druid/sql-1.json
reset-all.json Reset all stat
